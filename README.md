# travel-planner
BCA academic project

A full-stack Travel Planner web application developed as a BCA final year academic project.
This application helps users plan trips, manage destinations, and organize travel details.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: MongoDB
- Tools: VS Code

## Features
- User authentication
- Create and manage travel plans
- Destination details
- Backend API integration
- Responsive UI

## Project Structure
- frontend/  → User Interface
- backend/   → Server-side logic and APIs

## How to Run the Project
1. Clone the repository
2. Run npm install inside the backend folder
3. Create .env using .env.example
4. Start server using node server.js
5. Open frontend/index.html in browser
